## Features（本 release 包含）

### v0.2 基线（保留）

- W 轴切片、关卡图、切片实体、tap 移动 + W dial
- LevelSelect 选关、LevelCatalog、Completed 后 **N** 下一关
- macOS standalone 构建（`WSliceProto/builds/macos/W-Slice.app`）

### v0.3 新增 — Demo polish（PR #13–#14）

- **完成/失败 overlay** — `LevelOutcomeOverlayView`：Next / Retry / Level Select 可点击
- **Playing 状态下 **R** 重开** — `LevelRestartRules` / `LevelSession` 扩展
- **每关教学提示** — `LevelTutorialController` + `LevelTutorialDismissRules`（W 变化 / 移动 / 交互后消失）
- **LevelSelect demo 首页** — 标题、副标题、关卡主题 hint、**v0.3.0** 版本号、Quit

### v0.3 新增 — Authoring hardening（PR #15–#16）

- **LevelCatalog 校验器** — LevelId 唯一、Build Settings 顺序、Definition 对齐
- **Graph mutation 规范化** — `GraphMutationModel` + `LevelGraphMutationController`；Gate 拉杆解锁可 restart 回滚
- **Interactable profile** — `WInteractableProfile`；失败 tap 显示 W 区间 hint
- **GrayboxLevelRecipe** — 共享 movement / snap / UI / profile 目录常量

## Known Limitations

- **CI（可选）** — `.github/workflows/wslice-validate.yml` 提供 L0/L1；需在 repo secrets 配置 `UNITY_LICENSE`、`UNITY_EMAIL`、`UNITY_PASSWORD`
- **batchmode 测试不稳定** — `-runTests` 有时 exit 0 但不产出 XML
- **仅 macOS** — 无 Windows / Linux standalone
- **无签名公证** — `.app` 未 notarize
- **无正式美术/音效** — 灰盒 demo
- **无存档** — 无进度持久化
- **五关 graybox demo** — 美术与音效仍为占位
