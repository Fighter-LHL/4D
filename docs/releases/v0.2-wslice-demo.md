## Features（本 release 包含）

### 核心机制

- W 轴切片：`WState`、`WRange`、`WSnapResolver`、平滑插值与 snap
- 关卡图：`LevelDefinition` + BFS 路径 + W 门控边
- 切片实体：`SliceProfile` + Presenter（Fade / Scale / Shader）
- 玩家：tap 移动、W dial、W-aware movement retry

### 关卡与流程

- **三关 demo：** Garden_01 → Platform_01 → Gate_03
- **LevelSelect** 选关入口（Build Settings 第一项）
- **LevelCatalog** + **LevelFlowModel**：Completed 后 **N** 下一关
- **LevelSession** 生命周期：NotStarted / Playing / Completed / Failed / Restarting
- 目标检测与 **R** 重开（仅 Completed / Failed）

### 关卡机制摘要

| 关卡 | 机制 |
|---|---|
| Garden_01 | W 门控缺口与隐藏楼梯 |
| Platform_01 | `OffsetBridge` 随 W 上升，West→East 仅在 W∈[0.45,0.65] 可通行 |
| Gate_03 | W 门控拉杆解锁 GateRoom→Goal；移动中 W 关闭路径 → Failed |

### 交互

- `IWorldInteractable` + `SliceInteractionModel`
- Tap raycast：先 interactable，再移动
- `NotInteractiveAtCurrentW` HUD 反馈

### 构建与视觉

- macOS standalone：`./scripts/build-macos.sh` 或 `WSlice → Build/macOS Standalone`
- 统一输出：`WSliceProto/builds/macos/W-Slice.app`
- 构建 manifest：`WSliceProto/builds/macos/build-info.json`
- 灰盒 URP Lit 材质统一（`GrayboxVisual`）

## Known Limitations

- **无 CI** — GitHub Actions 未配置；验证依赖本地或 PR 手动记录
- **batchmode 测试不稳定** — `-runTests` 有时 exit 0 但不产出 XML
- **重开限制** — **R** 仅在 Completed / Failed 生效，Playing 中无效
- **仅 macOS** — 无 Windows / Linux standalone
- **无签名公证** — `.app` 未 notarize，首次打开可能需系统安全确认
- **无正式美术/音效** — 灰盒 demo
- **无存档** — 无进度持久化、无关卡解锁 gating
